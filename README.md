# Battleship Card Game (Java)

Study project developed in Java using Processing/PApplet.

The project implements a multiplayer Battleship card game with a graphical user interface and network communication between two game instances.

Features:
- Client/server architecture
- Multithreading for communication and updates
- GUI implementation using Processing (PApplet)
- Turn-based game logic
- Card-based battleship mechanics
- JShell playable version

Technologies:
- Java (JDK 17)
- Processing / PApplet
- controlP5

Note:
This project was developed during a university course and documents my learning progress in object-oriented software development and interactive systems.
